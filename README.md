# gitstamps
Group Project 3, WDI7 General Assembly

### Install locally
1. Fork and clone this repo
2. cd into directory
3. `npm install`
4. start instance of `mongod`
5. run node db/seeds.js
6. run `nodemon index.js`
7. view on `localhost:3000`
